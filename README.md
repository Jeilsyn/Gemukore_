 Descripción del Proyecto

Gemukore es una red social innovadora diseñada específicamente para jugadores de videojuegos. Nuestra plataforma facilita la comunicación y proporciona acceso inmediato a información clave sobre juegos.
---
## Objetivo Principal

    Conectar a los jugadores y brindarles un espacio accesible para adentrarse en el mundo gamer sin presiones y sin miedo

#✨ Funcionalidades Principales
🤝** Interacción Social**

    Comunidad de jugadores - Sistema de amigos y seguidores

    Matchmaking inteligente - Encuentra compañeros de juego ideales

    Chat en tiempo real - Comunicación fluida entre usuarios

    Sistema de moderación - Ambiente seguro y positivo

🎮 **Contenido Personalizado**

    Recomendaciones inteligentes - Basadas en tus intereses y preferencias

    Exploración de géneros - Descubre nuevos juegos sin restricciones

    Biblioteca personal - Organiza tus juegos favoritos

📚 **Aprendizaje y Desarrollo**

    Tutoriales interactivos - Guías paso a paso para principiantes

    Base de conocimiento - Comandos, mecánicas y estrategias

    Guías de personajes - Información detallada sobre cada personaje

    Foros especializados - Discusiones técnicas y estrategias
Aplicación web desarrollada como parte de un Trabajo de Fin de Grado (TFG).

El proyecto combina un **frontend** en React desplegado en Netlify, un **backend** en Node.js alojado en Render.com, y conexión con Appwrite (BaaS) para la gestión de datos.

---

### 🔗 Versión en línea

* **Frontend:** [gemukoree.netlify.app](https://gemukoree.netlify.app/)
* **Backend API:** [gemukore-backend.onrender.com](https://gemukore-backend.onrender.com/)

---

### 🚀 Características principales

* Interfaz web moderna construida con **React**.
* **Backend** en **Node.js/Express**, desplegado en Render.
* Comunicación entre frontend y backend mediante **API REST**.
* Integración con **Appwrite** como servicio BaaS.
* Despliegue automático en **Netlify** (frontend) y **Render** (backend).

---

### 📦 Requisitos previos

Antes de instalar el proyecto en local, asegúrate de tener:

* **Node.js** (incluye npm).
* **Conexión a Internet** (la app depende de Appwrite para funcionar).

---

### ⚙️ Instalación y configuración local

1.  **Clonar el repositorio:**

    ```bash
    git clone [https://github.com/Jeilsyn/Gemukore_.git](https://github.com/Jeilsyn/Gemukore_.git)
    cd Gemukore_
    ```

2.  **Instalar dependencias:**

    ```bash
    npm install
    ```

    Esto generará la carpeta `node_modules` con todas las dependencias necesarias.

3.  **Levantar el backend** (desde la carpeta `backend`):

    ```bash
    cd backend
    node index.js
    ```

    Deberías ver en consola: `Servidor backend corriendo en http://localhost:3002`

4.  **Levantar el frontend** (desde la carpeta principal del proyecto):

    ```bash
    npm start
    ```

    El frontend se ejecutará en `http://localhost:3000`.

---

### 🌍 Despliegue en producción

* **Frontend:** Netlify (sin instalación necesaria).
* **Backend:** Render.com.

El frontend se comunica directamente con la API desplegada en Render para realizar operaciones como solicitudes `POST` y mostrar las respuestas en la interfaz.
